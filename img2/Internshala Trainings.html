QUESTION 4
Multiple choice question
(3/3 MARKS)

If a <div> element is explicitly given a height 70px, a width 120px, border-width 5px, padding 20px, how much square pixel of space this <div> element will take when the box-sizing property is set to content-box and to border-box?

A.  

content-box: 22400, border-box: 8400

B.  

content-box: 20400, border-box: 8400

C.  

content-box: 18400, border-box: 7200

D.  

content-box: 16400, border-box: 7200

Bravo! Correct answer.
Solution

Correct answer : B

Your answer : B

Explanation

When the box-sizing is set to content-box, the height and width given is set for the content part only - the border and padding will take additional space.

The <div> will take a width of 120px + 2*5px + 2*20px = 170px.

The <div> will take a height of 70px + 2*5px + 2*20px = 120px.

So, the total space the <div> element will take is 170px*120px = 20400 sq px.

And, when the box-sizing is set to border-box, the height and width given is set for the entire <div> element from border to border - the border and padding will not take any additional space.

The <div> will take a width of 120px and a height of 70px as given.

So, the total space the <div> element will take is 120px*70px = 8400 sq px.

QUESTION 11
Multiple choice question
(3/3 MARKS)

What will be the output of the following PHP code?

<?php
$i = 0;
for($i++; $i++; $i++) {
    echo $i . " ";
    if ($i == 6) {
        break;
    }
}
?>

A.  

2 4 6

B.  

1 3 5 7 9 ... continues to print infinitely

C.  

1 2 3 4 5 6

D.  

Syntax Error

Excellent! Correct answer.
Solution

Correct answer : A

Your answer : A

Explanation

The for loop has three parts: initialization, condition, iteration. Each of these parts can technically contain any valid expression. The initialization executes just once before the start of the loop, which makes $i as 1. The condition part is executed at every iteration at the entry level. This makes $i as 2 in its first iteration. From the next iteration every time, the condition and the iteration gets executed, incrementing $i by 2. The loop breaks as the variable becomes 6.
